Veloca Manifesto (Light)

Writing should be completely frictionless. Veloca is built on a simple philosophy: the interface should dissolve, leaving only you and your typography.

"Perfection is achieved, not when there is nothing more to add, but when there is nothing left to take away."

Premium Light Mode

Unlike standard light modes that blast pure white into your eyes, we balance #FFFFFF with precise layers of #F4F4F5 and incredibly subtle #E4E4E7 borders.

// Shadcn Light Theme Config
const config = {
  theme: 'light',
  colors: {
    background: '#ffffff',
    foreground: '#09090b',
    border: '#e4e4e7'
  }
}

Everything you need, nothing you don't.

Built with Shadcn design principles to keep you in the flow.

Agentic Intelligence

Select any text and invoke the AI. The agent seamlessly writes, edits, and calls tools inline without breaking your typography.

Git Native

Built-in version control just like VS Code. See modified badges, stage files, and commit directly from the sidebar.

Obsessive Typography

Perfectly tuned line-heights, negative letter-spacing for headings, and native Mac/Windows font stacks. It's beautiful by default.
