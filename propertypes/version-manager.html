Veloca Manifesto

Writing should be completely frictionless. Veloca is built on a simple philosophy: the interface should dissolve, leaving only you and your typography.

"Perfection is achieved, not when there is nothing more to add, but when there is nothing left to take away." — Antoine de Saint-Exupéry

Design Language

We strictly adhere to a monochrome color palette. No gradients, no messy shadows. Just stark #000000, crisp #FFFFFF, and perfectly measured grays for borders.

// Shadcn configuration for Veloca
const config = {
  theme: 'dark',
  radius: '0.5rem',
  colors: {
    background: '#09090b',
    foreground: '#fafafa',
    border: '#27272a'
  }
}

Typographic Scale

Like Typora, Veloca is essentially a live-preview markdown editor. However, we bring the premium Shadcn UI aesthetic into the text area. Notice the letter spacing on the headings, the font choices (Inter & JetBrains Mono), and the structural line-heights.
