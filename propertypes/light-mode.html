Veloca Manifesto (Light)

Writing should be completely frictionless. Veloca is built on a simple philosophy: the interface should dissolve, leaving only you and your typography.

"Perfection is achieved, not when there is nothing more to add, but when there is nothing left to take away."

Premium Light Mode

Unlike standard light modes that blast pure white into your eyes, we balance `#FFFFFF` with precise layers of `#F4F4F5` and incredibly subtle `#E4E4E7` borders.

// Shadcn Light Theme Config
const config = {
  theme: 'light',
  colors: {
    background: '#ffffff',
    foreground: '#09090b',
    border: '#e4e4e7'
  }
}

Click the Settings button in the bottom left, or test the Toast Notifications in the top right header.
